# Projet_Kahoot
Projet pour l'ACS (projet en cours)

## Description  
Consignes du projet : 

Réaliser un quiz Kahoot
 Responsive
* Système de points
* Connexion à plusieurs via un code Pin et pseudo unique
* Timer pour les questions
* Site en ligne via FTP
* Animations
* Design moderne

## Langages et outils utilisés

* HTML5
* CSS3
* Javascript
* PHP
